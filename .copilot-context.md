# Project Context

This is a personal developer portfolio built with:

- Next.js (App Router)
- Tailwind CSS
- shadcn/ui (Radix)
- Framer Motion
- next-intl (i18n)

Goals:
- Minimalist, modern, dark UI
- Single-page layout with sections
- Clean architecture and maintainable code
- Subtle, professional animations only

Sections:
- Hero
- About
- Projects (placeholder for now)
- Skills
- Contact

Constraints:
- Keep components small and reusable
- Avoid overengineering
- Prefer clean Tailwind over complex abstractions
- Use i18n for all visible text
